# Get Started

OpsOrchestra helps you transform scattered documents and data into actionable intelligence. Here's how to get started.

## For SaaS Users

<Steps>
  <Step title="Sign In">
    Navigate to your OpsOrchestra dashboard and log in with your credentials.
  </Step>
  <Step title="Upload Documents">
    Upload your first document (PDF, CSV, or Excel). OpsOrchestra automatically:
    - Extracts text and tables
    - Identifies entities (companies, people, metrics, tools)
    - Maps relationships to your knowledge graph
  </Step>
  <Step title="Connect Data Sources">
    Optionally connect your databases for live data sync:
    - PostgreSQL
    - MySQL (coming soon)
    - More connectors planned
  </Step>
  <Step title="Ask Questions">
    Use natural language to query your data:
    - "What are our top vendors by spend?"
    - "Show me KPIs for Q4"
    - "How does vendor X compare to Y?"
  </Step>
  <Step title="Review Insights">
    Explore the Tactical HUD for:
    - KPI tracking and trends
    - Anomaly detection
    - Automated reports
  </Step>
</Steps>

## For Developers (Self-Hosted)

<Accordion title="Local Development Setup">

### Prerequisites
- Python 3.11+
- Node.js 18+
- Anthropic API key

### Installation

```bash
# Clone the repository
git clone https://github.com/im-sham/opsorchestra.git
cd opsorchestra

# Install Python dependencies
pip install -e .

# Configure environment
cp .env.example .env
# Edit .env and add your ANTHROPIC_API_KEY
```

### Start the Backend

```bash
uvicorn app.main:app --reload
```

### Start the Frontend

```bash
cd web
npm install
npm run dev
```

### Access Points

| Service | URL |
|---------|-----|
| API | http://localhost:8000 |
| API Docs | http://localhost:8000/docs |
| React UI | http://localhost:5173 |

</Accordion>

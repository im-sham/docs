# OpsOrchestra

> Transform operational chaos into actionable intelligence. OpsOrchestra is an AI-first platform that builds institutional knowledge with every interaction.

## The Problem

Operations teams drown in scattered documents, disconnected data sources, and tribal knowledge. Critical insights hide in spreadsheets, PDFs, and databases while decisions get made on gut instinct.

## The Solution

OpsOrchestra combines conversational AI with enterprise-grade intelligence:

## Core Capabilities

| Capability | Description |
|------------|-------------|
| **Document Intelligence** | Ingest PDFs, CSVs, Excel with automatic chunking and entity extraction |
| **Knowledge Graph** | LLM-powered relationship discovery between companies, people, metrics, and tools |
| **Data Connectors** | Live sync from PostgreSQL, MySQL, and other databases |
| **Hybrid RAG** | Intelligent query routing between semantic search and graph expansion |
| **KPI Analytics** | Metric extraction, trend analysis, and anomaly detection |
| **Decision Tracing** | Full audit trail with OpenTelemetry-compatible trace IDs |

## Tech Stack

| Layer | Technology |
|-------|------------|
| Backend | FastAPI |
| Agent | LangGraph + Hybrid RAG Router |
| LLM | Claude 3.5 Sonnet |
| Vector Store | ChromaDB |
| Graph | NetworkX |
| Observability | Langfuse |
| Frontend | React 19 + Vite |
